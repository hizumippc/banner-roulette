# banner-roulette# バナー練習用 お題ルーレット

バナー制作の練習用に作成した、お題自動生成ツールです。
ランダムに「サービス・ターゲット・目的・トーン・訴求・サイズ」を組み合わせて、実践的なお題を生成します。

---

## 🎯 使い方

1. 「ルーレット」ボタンを押す
2. 表示されたお題に沿ってバナーを制作
3. 必要に応じて個別に再抽選（🔄ボタン）も可能
4. X共有ボタンでお題をそのまま投稿可能

---

## 💡 特徴

* 実務に近い粒度の「サービス単位」でお題を生成
* ターゲット・目的・トーンなども自動で組み合わせ
* 不自然な組み合わせは出ないように制御（例：学生×投資 など）
* 履歴機能で過去のお題も確認可能
* シンプルなUIで短時間トレーニングに最適

---
